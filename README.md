# ReeseAlgoTrade

ReeseAlgoTrade is a Python class that utilizes historical market data, calculates technical indicators, and trains a machine learning model (HistGradientBoostingClassifier) for educational and informational purposes.

## Usage

1. Instantiate ReeseAlgoTrade:

```python
reese_algo_trade = ReeseAlgoTrade()

reese_algo_trade.fetch_historical_data()

reese_algo_trade.calculate_technical_indicators()

reese_algo_trade.train_machine_learning_model()


Requirements Python 3.11 xnumpy pandas requests ta scikit-learn
